Kalah Bot

Interactive game-tree explorer and play interface for Kalah(6,4). React frontend over a Python alpha-beta solver with a 15-stone endgame tablebase.

Note: this hosted demo renders the precomputed initial game tree (first 4 plies). Tree-node expansion and the Play vs P1 mode rely on a local Python backend (the alpha-beta solver and a 15-stone endgame tablebase, ~800 MB) and are not available here.
