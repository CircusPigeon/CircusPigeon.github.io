Kalah Bot

I made a 3D game tree visualization for the board game Kalah (see https://en.wikipedia.org/wiki/Kalah for more info). Every node represents a game state, and the positions are evaluted using a depth 12 minimax. I also designed a very strong bot for player 1 which only looks 2 moves ahead. It is beatable, though! Kalah was weakly solved in 2000, and it is a win by 8 for player 1 with perfect play.
